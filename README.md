1. Player **X** starts.
2. Each player taps any empty cell to mark it (X or O).
3. Win happens when:

   * 3 symbols match in a row
   * 3 symbols match in a column
   * 3 symbols match diagonally
4. If all cells fill with no winner â†’ **Draw**
5. Click **Restart** to play again.
